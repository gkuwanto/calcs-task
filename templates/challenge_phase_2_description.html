Testing Phase

This is the final evaluation phase where participants submit their best system for evaluation on the hidden test set, which includes both seen and unseen language pairs.

Phase Details

  • Duration: February 15, 2025 (24 hours only)
  • Submission Limit: 1 submission only
  • Evaluation: Systems will be evaluated on both public and private test sets
  • Final Ranking: Based on weighted performance across all language pairs

Important Notes

  • Choose your final submission carefully as only one submission is allowed
  • The private test set includes previously unseen language pairs
  • Final results will be announced after the completion of this phase
