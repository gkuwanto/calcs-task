Submission Guidelines

File Format

Please submit your predictions in a CSV file with the following columns:

  • id: The unique identifier for each test instance
  • predicted_choice: Your system's prediction (must be one of: "sent_1", "sent_2", or "tie")

Example Submission Format:

id,predicted_choice
1,sent_1
2,sent_2
3,tie
4,sent_1
...

Submission Rules

  1. Development Phase: Maximum 5 submissions per day
  2. Test Phase: Only 1 final submission allowed
  3. All submissions must be in CSV format
  4. Each submission must include predictions for all test instances

Important Notes

  • Ensure your CSV file has the correct column names and format
  • Invalid formats or missing predictions will result in submission failure
  • The system will automatically validate your submission format before evaluation
