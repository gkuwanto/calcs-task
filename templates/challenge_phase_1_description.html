Development Phase

This is the development phase where participants can test and improve their systems using the provided development set. The development set includes examples from English-Hindi, English-Tamil, and English-Malayalam language pairs.

Phase Details

  • Duration: January 17, 2025 - February 14, 2025
  • Submission Limit: 5 submissions per day
  • Feedback: Immediate feedback on accuracy scores
  • Leaderboard: Public leaderboard updated in real-time

Development Set

The development set contains labeled examples that can be used to train and validate your systems. Use this phase to experiment with different approaches and improve your system's performance.
